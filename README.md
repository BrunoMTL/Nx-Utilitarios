### Projetos Técnicos
**Nx Utilitários (C#)**  
GitHub: github.com/BrunoMTL/Nx-Utilitarios  
Automatização de workflows de NX usando NXOpen, com foco em geração de processos, validação automática e abstração de regras de negócio. Projeto demonstra habilidades em engenharia de software aplicada, integração com APIs de automação e construção de ferramentas utilitárias para sistemas complexos.
